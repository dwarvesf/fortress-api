# On Leave Form Page

This directory contains a redirect page to the Notion on-leave form.

## Deployment

Deploy to Cloudflare Pages:

```bash
npx wrangler pages deploy . --project-name=onleave-form --commit-dirty=true
```

The page will redirect users to: https://www.notion.so/dwarves/344c714343674d6cb6fd49fabeb43ce3
