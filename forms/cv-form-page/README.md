# CV Form Page (Refer a Candidate)

This directory contains a redirect page to the Notion refer-a-candidate form.

## Deployment

Deploy to Cloudflare Pages:

```bash
npx wrangler pages deploy . --project-name=cv-form --commit-dirty=true
```

Custom domain: `cv.d.foundation`

The page will redirect users to: https://www.notion.so/dwarves/6fba99cb38554d3eba407fad661db53c
